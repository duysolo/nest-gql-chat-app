# nest-gql-chat-app
Demo chat app built on top of NestJS &amp; Apollo GraphQL
- Medium posts: https://medium.com/@duysolo/build-a-chat-app-using-nestjs-fastify-apollo-graphql-nextjs-4b3999dd35ad

## Run NestJS server
```
cd backend;
npm ci;
npm run start
```

## Run NextJS frontend
```
cd ui;
npm ci;
npm run dev
```
